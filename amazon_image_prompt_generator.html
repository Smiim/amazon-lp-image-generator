    メイン画像 — 必須要件

    • 背景は純白（RGB 255,255,255）
    • 商品がフレームの85%以上を占める
    • 実際の商品を正確に表現する
    • 最小1000×1000px（推奨2000×2000px）
    • JPEG / PNG / TIFF / GIF 形式
    • sRGB または CMYK カラーモード
    • テキスト・ロゴ・透かしの追加
    • 白以外の背景・グラデーション
    • 商品以外の小道具・飾り
    • パッケージのみで商品不可視

    サブ画像 — 推奨・禁止

    • ライフスタイル・使用シーン OK
    • 商品特徴を説明するテキスト OK
    • 複数角度・詳細アップ OK
    • インフォグラフィック・比較表 OK
    • モデル・人物の使用 OK
    • 「送料無料」「セール」等の販促文字
    • Amazon ロゴ・バッジの使用
    • 在庫切れ・注文不可の表示
    • 不鮮明・ぼやけた画像
    • 成人向け露出・不適切コンテンツ

    技術仕様

    • 推奨サイズ: 2000×2000px（正方形）
    • 最小サイズ: 1000×1000px
    • 最大ファイルサイズ: 10MB
    • 推奨フォーマット: JPEG (.jpg)
    • カラーモード: sRGB
    • カルーセル最大: 9枚（メイン1+サブ8）
    • アニメーションGIF（動く画像）
    • 解像度500px未満の低品質画像
    💡 DALL-E 生成画像はデフォルト1024×1024px。Amazon使用は問題ありませんが、ズーム機能には2000px以上推奨。
